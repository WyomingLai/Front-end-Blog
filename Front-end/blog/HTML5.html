HTML5超連結、圖片與影音

路徑

路徑的表示方法方為絕對路徑及相對路徑
  • 絕對路徑：
    • <img src = "http://網址/檔案名稱">
  • 相對路徑：
    • 同層資料夾：
      <img src = "檔案名稱">
    • 上層資料夾：
      <img src = "../檔案名稱">
    • 下層資料夾：
      <img src = "資料夾名稱/檔案名稱">

超連結 < a href >

超連結須以 < a > 元素設定，語法如下： 
<a href="連結資源位置" target="連結開啟方式"> 顯示文字或圖片 </a>

屬性 說明
href
  1. 網址：網路網址或網頁必須加上通訊協定「http://」。
  2. 頁內：連結頁內中使用id屬性命名的元素位置，使用「#id」可以捲動到該id名的位置。
  3. 站內網頁：以目標頁面與目前頁面的相對路徑來連結到站內網頁「/下層資料夾/目標頁面」。
  4. 檔案：檔案超連結會開啟另存新檔對話框，需使用網路檔案絕對路徑或是站內檔案的相對路徑。
  5. 電子郵件：電子郵件會使用系統註冊之電子郵件軟體準備寄信，必須加上通訊協定「mailto:」，例如「mailto:YuMing@gmail.com」
  6. FTP：必須加上通訊協定「ftp://」，例如「ftp://ftp.YuMing.com.tw」
target
  1. "_blank"：以新的瀏覽器視窗開啟。
  2. "_parent"：開啟在目前視窗中，若有框架會在上一層框架中開啟。
  3. "_self"：開啟在目前視窗或框架中，是預設值。
  4. "_top"：開啟在目前視窗中，若有框架會移除框架以整頁顯示。
  5. "視窗名稱"：開啟在指定名稱的視窗中。

＊回到最上方的頁面超連結：若是頁內超連結所設定的id不存在，則會返回至頁面最上方。 因此許多人使用此方法來回到最上方，語法如下：

<a href="#top"> TOP </a>

表單 < form >

網頁裡的表單必須建立在< form >...< /form >之中，按下送出後會將表單中所有資料一併傳送到指定目的地處理。
在同一個頁面中可以包含多個表單，但表單中無法包含另一個表單。表單基本語法如下： 

<form action="資料傳送目的地(網址)" method="傳送方式"> .. </form>
    表單傳送資料的方式有「GET」及「POST」：
  • GET：瀏覽器會前往action指定的網址，並將表單中的資料附在網址之後。網址與資料間用「?」分隔，欄位資料則會用「&」分隔。
  • POST：瀏覽器會前往action指定的網址，並將表單中的資料放置在HTTP的表頭中傳遞。

文字、密碼及按鈕輸入欄位 < input >

輸入欄位對應的資料格式很多，所以有許多不同的欄位類型，< input >的基本語法如下： 

<input type="輸入欄位類型" name="欄位名稱" value="預設值">

屬性 說明
type
  • text：文字輸入欄。
  • password：密碼輸入欄，輸入值會以星號或實心圓顯示。
  • radio：單選鈕。
  • checkbox：多選核取方塊。
  • button、submit、reset：一般、送出、重置的按鈕。
  • hidden：隱藏欄位。
name 資料送出時會以「欄位名稱=資料值」的方式，因此欄位命名時建議要易於理解。
value 當沒有填寫資料即送出時，會以預設值做為傳送的資料值。
  • text：若要設定欄位長度或限制輸入資料長度可以使用size及maxlength屬性，例如： 
    <p><input type="text" name="username" size="20" maxlength="12"></p>
  • password：若輸入欄位是密碼即可使用，例如： 
    <p><input type="password" name="passwd" size="20" maxlength="12"></p>
  • radio：在多個選項中只能選一項，只要將name屬性設為相同，即會被視為同一組選項，例如： 
    <p>
    <input type="radio" name="gender" value="male">男
    <input type="radio" name="gender" value="female">女
</p>
  • checkbox：在多個選項中可以選擇一個以上的選項，只要將name屬性設為相同，即會被視為同一組選項，例如： 
    <p>
    <input type="checkbox" name="musictype" value="metal">金屬
    <input type="checkbox" name="musictype" value="hiphop">嘻哈
    <input type="checkbox" name="musictype" value="blues">藍調
    <input type="checkbox" name="musictype" value="classic">古典
</p>
  • button、submit、reset：完成表單填寫後，即可送出資料或重置將表單資料清空，其中value代表按鈕上顯示的文字，例如： 
    <p><
    <input type="submit" value="登入">
    <input type="reset" value="重置">
    </p>

＊不用 <p> .. </p> 包起來的話，帳號密碼會呈現在同一列。)


帳號：

密碼：

性別： 男 女

有興趣的音樂： 金屬 黑金屬 旋律金屬 前衛金屬 死亡金屬

輸入欄位的標籤 < label >

在輸入表單時常常不會精準的點在輸入欄位上，而是欄位名稱。如果使用< label >來顯示欄位名稱，當點選欄位名稱時也會進入輸入欄位。 若要使用< label >標籤，< input >就必須設定id屬性，因為< label >就是根據id屬性來對應，例如： 

<p>
    <label for="username">帳號：</label>
    <input type="text" name="username" id="username">
</p>
<p>
    <label for="passwd">密碼：</label>
    <input type="password" name="passwd" id="passwd">
</p>

下拉式選單 < select >、< option >

表單中若需要在多個項目中選取一個或多個選項時，使用radio或checkbox都會佔用較大的顯示區域。下拉式選單可以將選項折疊起來，是較理想的顯示方式。基本語法如下： 

<select name="欄位名稱">
    <option value="選項1回傳值">選項1</option>
    <option value="選項2回傳值">選項2</option>
      ...
    <option value="選項n回傳值">選項n</option>
</select>

屬性 說明
selected 下拉式選單會以第一個選項為預設選項，若要使用其他選項為預設，可以在該選項<option>中加入selected屬性，如下： 
<select name="欄位名稱">
    <option value="選項1回傳值">選項1</option>
    <option value="選項2回傳值 selected">選項2</option>
    <option value="選項3回傳值">選項3</option>
</select>
size 下拉式選單預設只會顯示一個項目，若要一次顯示多個項目，可以在<selected>中加入size屬性，如下： 
<select name="欄位名稱 size="3">
    <option value="選項1回傳值">選項1</option>
    <option value="選項2回傳值">選項2</option>
    ...
    <option value="選項n回傳值">選項n</option>
</select>
multiple 若想讓下拉式選單中可以多選，可以在<selected>中加入multiple屬性，使用者即可在選取時按著Ctrl鍵 ( Mac要按Command ) 不放選取多個選項，如下： 
<select name="欄位名稱 size="4" multiple>
    <option name="選項1回傳值">選項1</option>
    <option name="選項2回傳值">選項2</option>
    ...
    <option name="選項n回傳值">選項n</option>
</select>

HTML5 新增的表單元件與屬性

注意！各家瀏覽器支援的程度不一，但隨著HTML5的普及，應該很快能在各個平台上執行。
HTML5新增的屬性：

元件 說明
搜尋輸入欄：不同瀏覽器會有不同功能。
<input type="search">
電子郵件欄：必須輸入電子郵件格式，否則無法送出。
<input type="email">
URL輸入欄：必須輸入URL格式，否則無法送出。
<input type="url">
電話輸入欄：在手機上使用會顯示數字鍵盤。
<input type="tel">
數字輸入欄：只能輸入數字，旁邊會有增減按鈕。
<input type="number">
日期輸入欄：會顯示年/月/日，可以使用小日曆選取。
<input type="date">
範圍輸入欄：可以拖曳的移動軸。
<input type="range">
顏色輸入欄：可以開啟顏色盤以選取顏色。
<input type="color">


HTML5新增的屬性：
屬性 說明
autofocus 網頁開啟後會自動將輸入線一到該欄位中。每個頁面只能設定一個具有autofocus的欄位。 
<input type="text" autofocus>
autocomplete 依照歷史輸入值自動填入表單，預設為「on」，若要關閉可以設定為「off」。 
<input type="text" autocomplete="off">
placeholder 在欄位中顯示提示文字。 
<input type="text" placeholder="請輸入文字">
required 設定該元件為必填。 
<input type="text" required>
min/max/step 設定數字、日期、時間欄位的最小、最大值及遞增值。 
<input type="number" min="1" max="99" step="3">
pattern 設定正規表達式來規範輸入值的格式，適用在text、date、search、url、tel、email及password的欄位中。例如：
  • 輸入值第一個字必須為大小寫英文或數字：
    <input type="text" pattern="[a-zA-Z0-9]">
  • 輸入值前三個字必須為大小寫英文或數字：
    <input type="text" pattern="[a-zA-Z0-9]{3}">
  • 輸入值全部都必須為大小寫英文或數字：
    <input type="text" pattern="[a-zA-Z0-9]*">
  • 驗證輸入內容是否為電子郵件格式：
    <input type="email" 
    pattern="[a-z0-9._-+%]+@[a-z0-9.-]+\.[a-z]{2,4}$">
