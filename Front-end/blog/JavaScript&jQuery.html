認識JavaScript

在網頁中，HTML是建置網頁的基礎，CSS是網頁呈現的技術，而JavaScript則是網頁互動功能的關鍵。JavaScript是一種腳本式的程式語言，期代碼在客戶端執行前 不需要編譯，而是將文字格式的代碼傳給瀏覽器並由瀏覽器解譯執行。然而這種直譯式語言的安全性較差，每個人都能獲得原始碼內容。





建立JavaScript程式

<script>標籤

JavaScript是使用<script>標籤將程式碼加在HTML之中，瀏覽器只要讀取到<script>標籤便會利用直譯器來執行其中的程式： 

<script type="text/javascript">
...
</script>

＊ <script>中的type屬性是用來指定script的格式及類型，若是使用HTML5則可以省略type屬性。





JavaScript的基本語法與結構

基本語法規則

在編寫JavaScript時必須依循下列語法規定：

英文大小寫的不同：

在JavaScript中設定變數或常數時，英文大小寫是不同的。另外，JavaScript中的敘述及函式名稱都必須為小寫，否則無法執行該功能。

空白字元：

JavaScript會自動忽略多餘的空白字元。

結尾分號：

JavaScript中每一行結束時並無強制規定要加上「;」，但若要將多個敘述寫在同一行中時則需要以分號間隔。

註解符號：

JavaScript中使用「//」標示單行註解，也能用「/*.....*/」加上多行註解。

保留字：

在設定變數時應避開保留字。JavaScript常用的保留字有：break、default、function、return、var、case、void、debugger、new、throw、this、catch、const、finally、let、delete、switch..等。

變數命名與宣告


變數的宣告：

在JavaScript中是使用「var」來宣告變數，也能在單行之中宣告多個變數： 

var firstvar
var secondvar, thirdvar;


變數的命名：

JavaScript中對於變數的命名除了需要注意英文大小寫以及避開保留字以外，變數名稱的長度是沒有限制字數的、 變數名稱的起始字元只能使用英文或「_ 」符號、變數名稱裡不能使用「.」也無法用中文命名。
以下為幾個錯誤命名的範例： 

1var   //以數字開頭
變數1   //以中文命名
var.1   //不能使用「.」


設定變數值：

JavaScript中的變數資料可以是字串、數值、布林值..等。在宣告變數時可以用「=」指定變數的值： 

var name = "Banana博士"    //字串需以「"」包覆
var yellow = true
var length = 3.1415, unit = "meter"





JavaScript的運算子

算術運算子

JavaScript中的算數運算子適用一般四則運算的規則：加(+)、減(-)、乘(*)、除(/)。而其他還有餘數(%)、遞增(++)、遞減(--)等，如下： 

8%7    //輸出為1
a=1; a++;  //輸出為2
b=8; 8--;  //輸出為7

邏輯運算子

邏輯運算子會將算式兩邊進行比較，再將結果以布林值回傳。

邏輯運算子：     代表：當兩者相等時成立。

＊後來新增了「Object.is」運算子，和「===」一樣但會將「NaN」、「-0」和「+0」獨立處理，因此這三個不會相等。當Object.is(NaN, NaN) 會回傳true。而用「==」或「===」比較兩個NaN時則會回傳false。





JavaScript的流程控制

if / else 條件控制


if 單向選擇條件控制：

若只需要單項選擇的條件，即若滿足某條件則執行某程式內容，此為最單純的條件控制，基本格式如下： 

if (條件式) 執行程式的內容;
若執行程式內容不是單行，就必須用大括號將程式區塊包起來： 
if (條件式) {
      執行程式的內容;
}


if / else 雙向選擇條件控制：

當條件成立時執行某程式區塊，不成立時則執行另一個區塊： 

if (條件式) {
    條件成立時執行的程式內容;
}
else {
    條件不成立時執行的程式內容;
}
在JavaScript中可以使用「? :」來實現if/else的功能： 
(條件式) ? 條件成立時執行的程式內容 : 條件不成立時執行的程式內容;


else if 多項選擇條件控制：

if (條件式1) {
    條件1成立時執行的程式內容;
}
else if (條件式2) {
    條件2成立時執行的程式內容;
}
...
else {
    所有條件皆不成立時執行的程式內容;
}

switch 多項選擇條件控制：

switch會定義一個自訂變數，當該變數與下列某個case中的條件值相同時則執行該case的程式區塊： 

switch (自訂變數) {
    case 條件值1:
        自訂變數===條件值1時執行的程式內容;
        break;
    case 條件值2:
        自訂變數===條件值2時執行的程式內容;
        break;
    ...
    default:
        當自訂變數與所有條件值皆不相等時預設執行的程式內容;
}

＊自訂變數與條件值的資料型態要一致，default的程式區塊不是必填。

迴圈控制


while 迴圈：

while是最基本的迴圈，當符合條件時執行指定的程式，直到不符合條件時才跳出迴圈： 

while (條件式) {
    執行的程式內容;
}
do/while迴圈與while迴圈幾乎是一樣的，只差在前者會先執行迴圈內的程式，執行完後才判斷是否符合條件： 
do {
    執行的內容;
}
while (條件式)


for 迴圈：

for計次迴圈是先設定一個變數的初始值，再設定條件，最後是變數的計次方式。如下： 

for (設定變數初值; 條件式; 變數計次方式) {
    執行的程式內容;
}





JavaScript 函式、陣列與物件

function 函式

在攥寫程式時常遇到許多相同的動作，此時可以將這些經常使用或重複的程式整理成一個函式，在程式中隨時呼叫，這樣在閱讀或是維護上將會更為輕鬆。定義函式的基本語法為： 

function 函式名稱 ([參數1、參數2、...、參數n]) {
    執行的程式內容;
    [return 回傳值;]
}
在程式中呼叫函式的語法為： 
函式名稱 ([參數1、參數2、...、參數n]);

＊函式的命名規則與變數相同。當函式不需要回傳值時可以省略return。

array 陣列


建立一維陣列：

當程式中有大量同類型的資料需要儲存時，若宣告大量的變數，不但耗時且執行效率也不佳。而陣列是一群相同性質的變數集合，陣列中的每一份資料皆為陣列元素。 在預設狀況下，陣列會使用index(可以是整數或字串，預設為從0開始計算)來選取陣列中的元素。以下為範例： 

var album = new Array(11);
album[0] = "Coda";
album[1] = "LOST AND FOUND";
album[2] = "Answer is near";
...
album[10] = "Kimishidai Ressha";
也可以在建立物件時直接指派元素值： 
var album = new Array("Coda", "LOST AND FOUND", "Answer is near",...,"Kimishidai Ressha");
也可以直接使用[]符號來建立陣列： 
var album = ["Coda", "LOST AND FOUND", "Answer is near",...,"Kimishidai Ressha"];


建立多維陣列：

建立多維陣列的方式是先由一維陣列組成二維陣列，再架構到多維的陣列。例如： 

var album = new Array();
album[0] = ["Coda", "0:50"]
album[1] = ["LOST AND FOUND", "3:02"]
album[2] = ["Answer is near", "3:39"]
...
album[10] = ["Kimishidai Ressha", "4:14"]


取用陣列資料：

建立好陣列後，可以利用index來取得其中的值，以下為完整範例： 

<div id="ShowAlbumDetails"></div>
<script>
    var album = new Array();
    album[0] = ["Coda", "0:50"]
    album[1] = ["LOST AND FOUND", "3:02"]
    album[2] = ["Answer is near", "3:39"]
    album[3] = ["NO SCARED", "3:38"]
    album[4] = ["C.h.a.o.s.m.y.t.h.", "5:18"]
    album[5] = ["Mr.gendai Speaker", "3:53"]
    album[6] = ["Seken shirazu no unchuhikoushi", "3:31"]
    album[7] = ["Re:make", "3:24"]
    album[8] = ["Pierce", "4:24"]
    album[9] = ["Let's take it someday", "3:40"]
    album[10] = ["Kimishidai Ressha", "4:14"]

    //在目標元素中新建一個<table>
    document.getElementById("ShowAlbumDetails").innerHTML=
        "<table id='example-array'><tr><td>歌名</td><td>長度</td></tr></table>";

    //取得容器
    var Albumdetails = document.getElementById('example-array');

    for (i=0; i<album.length; i++) {
        //建立新的<tr>元素
        var Newtr = document.createElement('tr');

        for (j=0; j<album[j].length; j++) {
            //建立新的<td>元素
            var Newtd = document.createElement('td');
            //建立新的文字節點
            var Newnode = document.createTextNode(album[i][j]);
            //透過 appendChild 將 Newnode 加入至 Newtd
            Newtd.appendChild(Newnode);
            //透過 appendChild 將 Newtd 加入至 Newtr
            Newtr.appendChild(Newtd);
        }
        //透過 appendChild 將 Newtr 加入至 Albumdetails
        Albumdetails.appendChild(Newtr);
    }
</script>

object 物件

JavaScript是物件導向的程式語言，但在JavaScript中沒有class的概念，因為JavaScript中幾乎所有東西都是物件。物件是屬性跟方法 的組合，例如人有姓名、年齡、身高等，這些都是人的屬性 ; 而人可以唱歌、跳舞等這些便是人的方法。以程式來舉例：要得到上一節album陣列中的元素數量可以使用： 

album.length;    //length屬性的使用
而若要將指定的訊息顯示在頁面上或彈出式視窗，可以使用： 
document.write('日本的壓縮機');    //write()方法的使用
window.alert('非常稀少');         //alert()方法的使用

＊物件皆能以「.」來使用屬性或方法，而屬性跟方法的區分則是以後方有無()括號。


建立自訂物件：

var Person = {name:"Mr.Banana", age:69, height=87};
也可以將建立物件的動作化為函式： 
function Person(name, age, height) {
    this.name = name;
    this.age = age;
    this.height = height;
}
person1 = new Person("Mr.Banana", 69, 87);    //新增一個Person物件


建立物件方法：

var Person = {name:"Mr.Banana", age:69, height=87, 
    SelfIntroduction:function() {return "My name is "+name;}
};
也可以在建立物件函式中加入自訂的方法： 
function Person(name, age, height) {
    this.name = name;
    this.age = age;
    this.height = height;
    this.SelfIntroduction = function() {return "My name is "+name;}
};





DOM 文件物件模型

在DOM的標準下，文件中所有的標籤都是一個物件，這些物件以文件定義的結構形成一個樹狀結構。而在HTML中，每一個標籤及內容也可以被視為一個節點(Node)，各節點相連出一個樹狀結構。 因此只要善用DOM的概念就能在快速的在文件中找到要處理的部份進而存取。在JavaScript中可以使用getElementById()及getElementsByTagName()來找到HTML中指定的元素：

getElementById()

getElementById()是用來找到指定id屬性的元素： 

document.getElementById("id名稱")

getElementsByTagName()

getElementsByTagName()是用來找到指定的標籤名稱元素，因為網頁上相同的標籤名稱通常不只一個，所以回傳值會是一個陣列： 

document.getElementsByTagName("tag名稱")

範例：

<div id="albumDisplay"></div>
<script>
    var showTable = document.getElementById("albumDisplay")
    var albumTable = document.getElementById("example-array")
    var albumDetails = albumTable.getElementsByTagName("td")
    var msg = ""
    for(i=0; i<albumDetails.length; i++) {
        msg += albumDetails[i].innerHTML+"<br>";
    }
    showTable.innerHTML = msg
</script>
